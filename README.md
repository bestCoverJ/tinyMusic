# Vue2-fast

Vue2 快速启动框架

Vue.js 2 Quick Start Framework



⭐️框架已封装**axios**，**vuex**，**vue-router**



## 集成依赖 Integration dependencies

* vue 2.6.11
* element-ui 3.6.5

* vue-router 3.5.3
* vuex 3.6.2
* less 4.1.2
* axios 0.26.1
* js-cookie 3.0.1

`$ git clone https://github.com/bestCoverJ/vue2-fast.git`

`$ cd vue2-fast `

`$ yarn && yarn dev`
